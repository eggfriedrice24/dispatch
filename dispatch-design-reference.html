Design Principles

Warm, not cold
Better Hub uses cold zinc grays. We use warm tinted blacks and off-whites. The copper accent adds life. Your tool should feel like a well-made workshop, not a hospital.
Dense, not cramped
Information-dense like a terminal, but with intentional whitespace. Every pixel earns its place. The 42px navbar and 12px body text are deliberate — this is a power tool.
Editorial serif meets engineering mono
Instrument Serif for display type gives personality. JetBrains Mono for code. DM Sans for everything between. The mix says "crafted by engineers with taste."
Keyboard-first, always
Every action has a keybinding. Navigation is j/k. Actions are single letters. The UI shows shortcuts inline, not hidden in docs. This is a desktop app — act like it.
